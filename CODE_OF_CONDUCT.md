# Code of Conduct

## Our Commitment

FinSight aims to maintain a respectful, inclusive, and collaborative development environment.

## Expected Behavior

- Respect all contributors.
- Provide constructive feedback.
- Focus on improving the project.
- Maintain professionalism.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Discrimination
- Offensive language

## Enforcement

Project maintainers reserve the right to moderate discussions and contributions when necessary.
