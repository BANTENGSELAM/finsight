# Definition of Done (DoD)

Sebuah Sprint dinyatakan selesai apabila:

## Engineering

- [ ] Kode berhasil dijalankan.
- [ ] Tidak ada error.
- [ ] Mengikuti MVC.
- [ ] Mengikuti Service Layer.
- [ ] Controller tetap tipis.
- [ ] Tidak ada business logic di Blade.
- [ ] Validasi menggunakan Form Request.
- [ ] Seluruh query aman (berbasis user).

## Quality

- [ ] Self Audit selesai.
- [ ] Repository Audit selesai.
- [ ] Architecture Score ≥ 95.
- [ ] Engineering Score ≥ 95.
- [ ] Tidak ada pelanggaran arsitektur.

## Documentation

- [ ] CHANGELOG diperbarui.
- [ ] PROJECT_STATE diperbarui.
- [ ] PROJECT_MEMORY diperbarui.
- [ ] ROADMAP diperbarui.
- [ ] README diperbarui jika ada perubahan fitur.
- [ ] Release Notes diperbarui jika mencapai milestone.

## Repository

- [ ] Git status bersih.
- [ ] Commit siap dibuat.
- [ ] Dokumentasi sinkron dengan implementasi.

Sprint **tidak boleh dianggap selesai** sebelum seluruh poin di atas terpenuhi.
