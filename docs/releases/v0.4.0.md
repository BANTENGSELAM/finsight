# Release v0.4.0 - Dashboard Analytics

**Date:** June 2026

## Overview
Release v0.4.0 introduces the core Analytics Dashboard to FinSight. This represents the completion of Sprint 4 and a major milestone in transitioning the platform from a simple tracker to a comprehensive financial insights tool.

## Key Features

### 1. Financial Summary
- Current Balance Calculation
- Monthly Income Aggregation
- Monthly Expense Aggregation
- Net Cash Flow Indicator

### 2. Visualization (Chart.js)
- **Income vs Expense:** A 6-month trend line chart to monitor overall financial growth.
- **Expense by Category:** A current-month doughnut chart to immediately identify spending habits.

### 3. Actionable Widgets
- **Recent Transactions:** A chronological list of the 10 most recent transactions.
- **Upcoming Subscriptions:** The 5 nearest upcoming active subscriptions to prevent unexpected charges.
- **Quick Actions:** Easy access to create new income, expenses, and subscriptions.

### 4. UI/UX & Design System
- Complete restructuring of the dashboard layout using Tailwind CSS Grid.
- Strict component extraction to maintain high modularity.
- Extracted domain partials for clean, maintainable presentation layers.

## Engineering Notes
- Zero N+1 queries introduced.
- Strict adherence to the Service Layer pattern (`DashboardService`).
- Inline asset loading leveraged for immediate charting availability.
